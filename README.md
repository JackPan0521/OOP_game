# 🎮 回合制戰鬥之我身處中世紀戰爭

一款用Java開發的回合制戰鬥遊戲，設定在中世紀戰爭時期。玩家需要在戰場中生存，通過戰鬥獲取經驗和榮譽，最終觸發不同的結局。

## 📋 遊戲特性

### 角色系統
- **玩家角色**：擁有HP、ATK、DEF等屬性，以及EXP、等級、榮譽、戰功、擊殺數等進階屬性
- **敵人分類**：小兵、將領、逃兵（三種難度敵人）
- **NPC**：傭兵（根據榮譽決定是否背叛）、流浪修道士（提供治療和預言）

### 戰鬥系統
- **回合制戰鬥**：玩家和敵人輪流進行行動
- **多種行動**：攻擊、防禦（傷害-30%）、逃跑（50%成功率）
- **動態傷害計算**：基於雙方ATK和DEF，包含20%的隨機波動

### 事件系統
- **戰場廢墟**：戰鬥後30%機率觸發
  - 搜刮物資 (HP +10, 榮譽 -10)
  - 幫助倖存者 (榮譽 +15)
  - 無視離開 (無變化)

- **投降敵軍**：WarPoint >= 50時觸發
  - 處決 (戰功 +10, 榮譽 -20, 擊殺數 +5)
  - 俘虜 (榮譽 +10)

- **教會審判**：榮譽 <= -30時觸發
  - 通過審判（40%成功率）
  - 失敗進入強制戰鬥

- **最後戰役**：戰功 >= 150時觸發（固定事件，遊戲結束）

### 結局系統（4種結局）

#### 🌟 榮譽騎士
- 條件：WarPoint >= 150 & 榮譽 >= 50 & 擊殺數 < 30
- 描述：以英勇和榮譽被銘記的偉大戰士

#### ⚔️ 戰爭倖存者
- 條件：WarPoint >= 150 & 榮譽 -20~49
- 描述：活著從戰場離開，被記憶但不特別

#### 🩸 嗜血屠夫
- 條件：WarPoint >= 150 & 榮譽 <= -50 & 擊殺數 >= 50
- 描述：沉溺於殺戮，成為戰爭的怪物

#### ☠️ 戰死沙場
- 條件：HP <= 0
- 描述：戰場上的犧牲者

## 🎯 遊戲流程

1. **初始化**：輸入玩家名字，開始遊戲
2. **日常選擇**：每天可以選擇：
   - 上陣線戰鬥
   - 查看狀態
   - 與NPC互動
   - 休息恢復HP
   - 結束冒險進行結局判定
3. **戰鬥**：進行回合制戰鬥，選擇攻擊、防禦或逃跑
4. **事件觸發**：根據條件自動觸發各類事件
5. **結局判定**：達成最後戰役條件後，遊戲進行結局判定

## 🎮 控制方式

遊戲採用文字菜單控制：
- 輸入數字選擇相應選項
- 根據提示進行遊戲操作

## 💡 遊戲建議

- 平衡榮譽和戰功以獲得更好的結局
- 謹慎對待敵人的投降選擇
- 定期休息維持健康的HP
- 與NPC互動可以了解可能的結局

- **繼承**：Character → Player/Enemy，NPC → Mercenary/WanderingMonk

- **封裝**：角色屬性的保護和訪問控制
- **抽象**：Character和NPC基類的設計
- **組合**：Player包含多個屬性和狀態
